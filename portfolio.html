Pearse's Portfolio Page

Hello and welcome to my portfolio. This haphazard collection of links is an example of things that I'm currently working on to some degree or other. A strong warning that everything is in some state of "work in progress".

Diplomacy Country Assignment

Inspired by setting up games of Diplomacy with friends, I wrote a python program to anonymously assign a country to each player. This page is web form version of that script, which lives on my github here. I'm terrified that the anonymous email functionality will get abused if I put it on the web so for now it's just for randomising people.

Birds as Gaeilge

I wrote a mastodon bot to post the names of birds in English and Irish. The bot takes a random entry from Colm Ó Caomhánaigh's Dictionary of Bird Names in Irish, finds its Wikipedia entry and posts the name of the bird in English, Irish and Latin to Mastodon along with the main image from Wikipedia. The script used to run once a day off a chronjob on my laptop, I may start it up again in the future.

Population Density of Ireland

There was a post on reddit.com/r/ireland where someone had taken Eurostat data and plotted the population density on the island of Ireland as a Joy Division (crab nebula pulsar) ridgeline plot. Arguments in the comments made me think "I can do better than that", so I did. Fortunately, I lost interest before engaging in an argument online.
